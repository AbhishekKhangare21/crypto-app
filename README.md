About Project cryptoverse

Description-
It shows the trending crypto currencies that are available in the market, with that latest
news related to that coins. It show all the information to that coin such as Total Market
Cap, Total Exchanges, Total Volume, and etc.
It fetches 2 different API using Axios and all the state is managed by Redux Toolkit., and
this app was built using the React JS and Ant Design.
Used Dynamic Routing for a single coin and show all the information & social links
related to that single crypto coin.
